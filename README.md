# diy_turtlebot

## Part Required for Mobile Platform

### Structure
1. Turtlebot3 Base (3D Printed)
2. M3 PCB Stand (screw & screw) 40mm

### Electronics
#### Controller
1. Arduino Mega 2560
2. Raspberry Pi 4 4GB
	1. SD Card 16GB
 
### Actuator/Sensor
1. DC Motor
	1. DC Motor Driver
	2. 12V DC Motor with Encoder
2. 6 DOF IMU
3. Camera
4. LiDAR

### Power
1. 12V LiPO Battery
2. Powerbank
